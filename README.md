# 3dSpanningTrees
Takes on .obj and returns the MST of it

Outputs an OBJ


## Installation

```
git clone <this>
java 3dSpanningTrees/src/main/SpanningTrees.java
```
***

Or run use the executable:

**For mac**
```
chmod +x run.command
```
Double click run.command

**For windows**

Double click run.bat
